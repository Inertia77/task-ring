# v21 Clarity Pass

## 改动目标

v20 的问题不是功能数量，而是视觉层级过重、颜色过散、按钮可读性不足。v21 主要做收敛：

- 顶部总控按钮改成深色主按钮，强化全局入口。
- 周计划池改成更稳定的白底卡片 + 单一蓝色主色，不再用大面积彩虹渐变。
- 周计划池任务卡恢复分任务显示；分任务状态按周保存。
- 修复 Italiano / Italian 这类语言任务被 `it` 关键字误判成 IT/AI 的问题。
- 游戏作战区降噪：不再使用过暗卡片，改为白底高对比结构。
- 周目标仍然使用安全弹窗，不在卡片里直接改数值。

## 周计划池分任务

周计划池任务如果配置了 `steps`，卡片内会显示「分任务 0/n」。
这些分任务按当前周记录，不再依赖某一天。

## 设计原则

- 主色：蓝 / 青，粉色只作为极少量强调。
- 卡片：白底、明确边框、轻阴影。
- 操作：开始计时必须高对比，不允许浅色按钮看不清。
- 页面：每个区域只表达一种意图，减少大面积装饰块。
