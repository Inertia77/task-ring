# V18 游戏作战区：今日清理 / 本周作战池

V18 把游戏作战区内部也拆成两类，但不会混进普通周计划池。

## 今日清理

用于每天或指定日要看的游戏任务，例如：

- 日常体力
- 咖啡 / 刮刮乐 / 委托
- 资料关注
- 某个指定星期检查的活动

这些任务在游戏作战区的「今日清理」里按星期显示。

## 本周作战池

用于本周完成一次即可的游戏任务，例如：

- 周常清理
- 深塔 / 海墟 / 全息
- 式舆 / 危局 / 鏖战
- 模拟宇宙 / 末日 / 虚构检查

这些任务只出现在游戏作战区的「本周作战池」，不进入普通「周计划池」。

## 编辑方式

打开总控 → 游戏任务编辑器：

- 上半区：当前星期的「今日/指定日清单」
- 中间区：每个游戏的「本周游戏作战池」
- 下方折叠区：游戏大卡设置

旧配置里的“周常、深塔、式舆、危局、模拟宇宙”等任务会自动归入本周池。
