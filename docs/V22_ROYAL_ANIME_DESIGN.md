# V22 Royal Anime Atelier Design Pass

## 方向
- 明亮二次元：保留轻盈、星光、收藏卡式质感，但避免暗黑电竞和高饱和彩虹噪声。
- 皇家精致风：珍珠白底、星冠金、淡紫蓝、细金线、徽章式按钮。
- 紧密排版：整体减少空隙、缩短按钮和卡片高度，保持原有信息密度，不删内容。

## 主要改动
- 新增 `assets/css/97-royal-anime.css` 作为最终覆盖层。
- `assets/css/main.css` 追加导入 v22 覆盖层，方便以后整体回退或继续迭代。
- 更新首页标题、锁屏文案、版本提示与页脚，使品牌方向统一为 Royal Atelier。
- 重绘 `assets/icons/favicon.svg`，从深色闪电改为珍珠底、星冠、任务环组合。

## 覆盖范围
- 全局背景、header、导航 dock、总控按钮、同步状态 pill。
- 今日执行环、任务表格、周计划池、游戏作战区、时间账本、资料库。
- GitHub 设置弹窗、任务/资料库/游戏编辑器、锁屏、toast、浮动计时器。

## 调整原则
旧 CSS 文件保持不大拆，v22 只作为最后一层覆盖。后续如果想换成其他皮肤，只需要替换或移除 `97-royal-anime.css` 以及 `main.css` 里的最后一行 import。
