# TASK RING v16：今日执行环 + 周计划池

## 核心变化

v16 把原来的「每日任务环」拆成两层：

1. **今日执行环**  
   只放真正需要今天出现的任务，例如每日保底、指定星期任务、遗留事项。

2. **周计划池**  
   放不必每天打卡、但希望本周持续推进的任务，例如语言、IT/AI、自然科学、创作、数据库整理等。

## 任务模式

每个任务新增 `plan_mode` 字段：

- `daily`：每日环。适合真每日保底任务。
- `scheduled`：指定日。只在选中的星期进入今日执行环。
- `weekly`：周计划池。不进入今日执行环，只按本周目标时间管理。

## 时间系统

- `estimated_minutes`：单次预计时长。
- `weekly_minutes`：本周目标时长。
- 历史时间日志同步到 `taskring-state.json`。
- 当前正在计时的秒表仍然只保存在本机，避免多设备混乱。

## 推荐用法

- 每天先看「今日执行环」。
- 有空再进「周计划池」，按目标缺口选择任务开始计时。
- 周末看「时间账本」，复盘时间实际流向。
